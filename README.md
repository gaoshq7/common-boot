## 简介

`common-boot`是一套针对于 Spring boot 2 研发环境下二次封装便于使用的基础环境API。

## 特性

- **自定义banner**：支持使用者提供图片作为程序启动banner。
- **动态加载bean**：提供程序启动时根据环境变量加载bean；提供程序启动后根据class path加载bean。
